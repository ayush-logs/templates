# Why does it exist?

Goal is to:

1. decrease decision fatigue when working on projects.
2. systemize repeated tasks.
3. get into the habit of articulating and creating systems for development.

---

# Example

1. Created a default `.gitignore` template file for DRF+ React project.
2. For every project copy paste the template.
3. Modify, if required.
